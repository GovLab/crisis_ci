Data Visualization and Mapping

By Kyle Pennell

Kyle Pennell, Solutions Engineer at CARTO, provides an overview of available data sources and tools for creating data visualizations about COVID-19. He demonstrates how CARTO Builder, along with other tools like Tableau Public and Datawrapper, can be used to make maps and other data visualizations. He also discusses how data visualizations have helped individuals and groups to better understand the pandemic and take action.
